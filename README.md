**INSTALL AND RUN**

To run project locally you need:

1) install `ruby-2.3.3`

2) install `postgresql`

2) run `bundle install`

3) copy `database.yml.example` to `database.yml` and fill in your db settings

4) run `foreman start -f Procfile.dev`
